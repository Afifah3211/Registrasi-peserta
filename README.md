# Sistem Registrasi Peserta

Proyek ini adalah aplikasi web sederhana untuk mengelola pendaftaran peserta. Aplikasi ini memungkinkan pengguna untuk menambah, melihat, memperbarui, dan menghapus data peserta (CRUD).

## 🚀 Fitur Utama
- **Tambah Peserta**: Input data peserta baru.
- **Daftar Peserta**: Menampilkan semua data yang telah terdaftar dalam bentuk tabel.
- **Update & Delete**: Mengubah informasi peserta atau menghapus data dari sistem.
